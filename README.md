# 🧠 EEG Data Analysis Hackathon

Welcome to the EEG Data Analysis Hackathon repository!
This repository serves as the central hub for data, code, and documentation related to the hackathon.

The goal of this hackathon is to collaboratively explore, analyze, and discuss EEG data using reproducible and open scientific practices. Participants are encouraged to experiment, learn from each other, and share ideas — regardless of prior experience with EEG analysis.

## 🎯 Objectives

- Gain hands-on experience with EEG data analysis
- Explore preprocessing, analysis, and visualization techniques
- Work collaboratively on shared datasets
- Practice reproducible and transparent research workflows
- Exchange ideas across disciplines and experience levels

## 🛠 Tools & Requirements

Most analyses will be demonstrated using:

- Python (e.g. MNE-Python, NumPy, SciPy, Matplotlib)
- Jupyter Notebooks

Please install required packages before the hackathon if possible.
A list of dependencies or an environment file may be provided in this repository.

## 🤝 How to Contribute

We encourage collaboration and sharing!

You can contribute by:
- Adding notebooks or scripts
- Improving documentation
- Sharing analysis ideas or visualizations
- Opening issues to ask questions or suggest improvements

If you’re unfamiliar with GitHub: don’t worry — facilitators will help, and you’re welcome to participate without committing code.

## 📏 Guidelines

- Be respectful and supportive of all participants
- Write clear and readable code
- Comment your analyses so others can follow your reasoning
- Aim for reproducibility rather than perfection

This hackathon is a learning environment — experimentation and mistakes are part of the process.

## 📢 Communication

Important information, updates, and discussions will be shared during the hackathon sessions and via the communication channels announced by the organizers.
